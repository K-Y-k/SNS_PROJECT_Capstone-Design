## :question: 어떤 프로젝트인가요?\(프로젝트의 목적\)
최근 주말 농장을 운영하는 중장년층이 늘고 있습니다. 

하지만 평일에 일을 하는 사람들은 주말에만 농장을 방문하기 때문에, 농장 관리 문제와 농작물 절도 문제가 주로 발생합니다. 
                   
위 문제들을 해결하기 위해 농사를 효율적으로 운영 및 관리에 도움을 제공하는 서비스를 기획한 프로젝트입니다.
